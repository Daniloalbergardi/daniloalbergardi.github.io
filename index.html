Criador de URL (com utm) para campanha de publicidade

Informe os parâmetros de campanha personalizados para gerar a URL.

Construa uma URL rastreável através das UTMs para que o Google Analytics e outras ferramentas possam rastrear o comportamento da sua campanha. Quando os usuários acessar sua URL com estes parâmetros, a ferramenta de web analytics registra e transforma dados em relatórios sobre a campanha de referência.

Você pode criar URLs com a ferramenta abaixo, basta adicionar as informações referente a sua campanha:

Preencha os campos obrigatórios (marcados com *), ao preencher, a URL completa da campanha será gerado.

* URL completa do site (Ex. https://www.example.com)

* Referência (Ex. google, facebook, newsletter, youtube)

* Ação (Ex. cpc, clique_banner, email)

* Produto, código promocional ou slogan (Ex. black_friday, promocao_abril)

Utilizado para diferenciar anúcios  

  • O URL gerado é atualizado automaticamente à medida que você faz alterações.
  • Não utilizar caracteres especiais e espaço entre palavras
